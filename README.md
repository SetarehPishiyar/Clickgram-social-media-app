# Clickgram

A modern social media platform inspired by Reddit, built with React, TypeScript, Tailwind CSS, React Query, and Supabase.

## Features

* 🔐 Authentication with Supabase
* 📝 Create and browse posts
* 🖼️ Upload post images
* 🏘️ Create and join communities
* 👍 Like and dislike system
* 💬 Threaded comments and replies
* ⚡ Real-time data fetching with React Query
* 🎨 Responsive dark UI with Tailwind CSS
* 📱 Mobile-friendly design

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router
* TanStack Query

### Backend

* Supabase Database
* Supabase Authentication
* Supabase Storage

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/clickgram.git
```

Move into the project:

```bash
cd clickgram
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Start the development server:

```bash
npm run dev
```
